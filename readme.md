# Nano

Nano is a statically typed programming language made for lightweight and (reasonably) fast programs. This repository holds the official bytecode interpreter for the language, and it's made in Lua(JIT). 

Some may say it's compiled, and others may say it's interpreted, but the ideal definition is that it's *bytecode interpreted*. Nevertheless, the section that turns the source code into the bytecode is a *compiler*.
Nano aims to have a reasonably performant runtime while being written in a language that makes it easy to contribute. You can still implement it in any other language if you want to.


  
> ⚠️  Nano is far from being production ready, which means you shouldn't use it yet!⚠️
